2 · Next.js (App Router)

Você já está usando convenções do Next: basta colocar os arquivos com estes nomes em app/:

app/
 ├─ icon.svg                    ← favicon.svg
 ├─ apple-icon.png              ← apple-touch-icon.png
 ├─ opengraph-image.png         ← opengraph-image.png
 └─ twitter-image.png           ← cópia do opengraph-image.png

O Next gera as <link> e <meta> automaticamente.

3 · Tipografia

A direção Camadas usa Space Grotesk (peso 500). Para o site, importe via Google Fonts:

<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;700&display=swap" rel="stylesheet">

Para uso fora do navegador (PowerPoint, vídeo, etc.), converta o texto dos SVGs em curvas no Figma/Illustrator.
